## Install

```sh
go get github.com/Ideolys/carbone-sdk-go
```

## Usage

```go
package main

import (
	carbone "github.com/github.com/Ideolys/carbone-sdk-go"
)

func main() {
	// ...
}
```

## Run tests

To run the tests, just run:

```sh
go test -v
```
